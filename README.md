- 副業の話はいつでもお待ちしております。
  - 特に2026年からは副業の稼動がなくなるため、現在は積極的に探しています。
  - 夜間・休日稼働が主になります。
    - 40-60時間/月程度が上限の想定です。
  - 単価は仕事内容にもよりますが、下限¥6,500〜くらいで探しています。
    - 稼働時間によって希望時給は変動します(稼働量が増えるほど金額は大きくなります)
    - 月給固定でのオファーは強く歓迎しております。
      - 仕事量に波がないこと前提
  - MTGなどは適宜想定稼働時間外でも対応します。
  - 仕事内容に関しては、「今後やりたいことについて」に沿っていることが望ましいですが、事業の制約はありません。
